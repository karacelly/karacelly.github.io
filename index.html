Rumus Luas Segitiga yaitu :

Di mana :

L = Luas

Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Di mana :

K = Keliling

S1 = Sisi A

S2 = Sisi B

S3 = Sisi C
